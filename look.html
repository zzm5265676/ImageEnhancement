RGB → HSV： V = max(r,g,b), S = Δ / V, H ∈ [0,6)
HSV → HVI： θ = πH / 3, Ck = sin(πV/2)1/k
坐标： Ĥ = Ck·S·cosθ，V̂ = Ck·S·sinθ
核对： ρ = √(Ĥ² + V̂²) = Ck·S
角度核对：θ = atan2(V̂, Ĥ) 与 πH/3 一致。
半径核对：ρ = √(Ĥ² + V̂²) 与 Ck·S 一致。
当前解释：当前点位于圆盘右侧，半径由 Ck·S 决定。
左列：RGB 输入与 k 中列：圆盘 + 一维 H 右列：数值 + 彩色/灰度相对值
